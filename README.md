# 极简地图

基于高德 JS API 的精简版地图工具，纯个人学习使用。

## 功能

- 🗺 地图浏览（无广告、无弹窗）
- 🔍 地点搜索
- 🛣 路线规划（驾车 / 公交 / 步行 / 骑行）
- 📍 当前定位
- 📱 移动端适配

## 使用方法

### 1. 申请高德 Key

1. 访问 [高德开放平台](https://lbs.amap.com/) 注册账号
2. 进入 [控制台](https://console.amap.com/) → 创建应用 → 添加 Key
3. 服务平台选择 **"Web端(JS API)"**
4. 2021年12月后申请的 Key 还需要安全密钥

### 2. 启动

直接浏览器打开 `index.html`，首次会弹窗要求输入 Key，输入后自动保存到 localStorage，后续无需重复输入。

### 3. 使用

- 左侧栏输入关键词搜索地点，点击结果跳转
- 路线规划模块选择出行方式，输入起点终点
- 点击地图任意位置可查看地址
- 点击定位按钮回到当前位置

## 技术栈

- 高德地图 JS API 2.0
- 原生 HTML / CSS / JS，无框架依赖
- 单页应用，本地直接打开即可运行
